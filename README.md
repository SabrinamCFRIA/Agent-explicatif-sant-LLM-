# Agent explicatif santé – LLM (Health Explainer)

## Description du projet
Ce projet présente un agent explicatif basé sur un modèle de langage de grande taille (LLM), conçu pour expliquer de manière pédagogique et structurée les concepts, enjeux et limites de l’intelligence artificielle appliquée au secteur de la santé.

L’agent est destiné à un usage informatif et académique. Il ne fournit aucun diagnostic médical et ne remplace pas l’expertise d’un professionnel de santé.

---

## Objectifs
- Comprendre le fonctionnement des modèles de langage de grande taille (LLM)
- Analyser leurs usages potentiels dans le secteur de la santé
- Identifier les risques techniques, éthiques et humains associés
- Proposer une démonstration responsable d’un agent IA en santé

---

## Fonctionnalités principales
- Explication de concepts liés aux LLM et à l’IA en santé
- Réponses structurées sous forme de points
- Ton neutre, prudent et pédagogique
- Intégration de garde-fous interdisant tout usage clinique
- Interface de démonstration via Streamlit

---

## Technologies utilisées
- **Langage** : Python 3
- **Modèle de langage** : Llama 3.2 1B (exécuté localement)
- **Serveur LLM** : LM Studio (API compatible OpenAI)
- **Interface** : Streamlit
- **Système d’exploitation** : Ubuntu
